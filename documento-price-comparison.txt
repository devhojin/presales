[H] 조사 한계 및 추가 확인 필요 사항
================================================================================

1. imweb 동적 렌더링으로 인해 WebFetch로 실제 상품 데이터 추출 불가
   (CSS/스타일 코드만 반환됨)
2. 대부분의 개별 상품 가격은 검색 캐시에서 확인 불가
3. 확인된 가격:
   - idx=194 (45억 시스템 구축): 정가 392,000원, 쿠폰가 89,000~99,000원
   - idx=134 (홈페이지 유지보수): 정가 392,000원, 할인가 99,000원
   - idx=154 (사업비 비목 산출표): 정가 10,000원, 할인 후 0원 (무료)
4. 전체 가격 범위: 99,000원 ~ 392,000원 (PPT 원본 기준)
5. 정확한 가격 확인을 위해서는 사이트에 직접 접속하여 확인 필요
   (브라우저에서 JavaScript 렌더링 필수)
