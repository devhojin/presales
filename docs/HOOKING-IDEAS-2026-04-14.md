# presales.co.kr 후킹·전환 강화 아이디어 리포트

작성일: 2026-04-14
대상: 공공조달 제안서 마켓플레이스 presales

---

## 1. 첫 인상 후킹 (Above the Fold)

### 1-1. 실시간 사회적 증거 (Social Proof)
- 홈 히어로 우측 하단에 "오늘 23명이 기술제안서를 구매했어요" 롤링 토스트
- 최근 24시간 구매/다운로드 건수를 Supabase에서 집계해 10초마다 순환
- 구현: `SELECT count(*) FROM orders WHERE status='paid' AND created_at > now()-interval '24 hours'`

### 1-2. 긴박감 유발 (Scarcity/Urgency)
- 공고 마감 임박(D-3 이내) 배너: "오늘 마감 공고 5건 · 지금 제안서 확인"
- 상품 상세: "최근 1시간 내 3명이 장바구니에 담음"
- 쿠폰 만료 타이머: `WELCOME10K` 24시간 카운트다운

### 1-3. 고유가치 한 줄 헤드라인 교체
- 현재: "공공조달 기술제안서 마켓플레이스"
- 제안: **"입찰 실패한 제안서 말고, 실제 낙찰된 제안서를 참고하세요"**
- 신뢰 배지: "누적 다운로드 1,200+ · 평균 평점 4.7"

## 2. 비회원 → 회원 전환 후킹

### 2-1. 미리보기 PDF 무료 제공 (1-page teaser)
- 상품 상세에서 **비회원도 1페이지만 워터마크 포함 미리보기** 가능
- 2페이지 이상은 로그인 유도: "나머지 23페이지는 무료 회원가입 후 열람"
- 효과: 가치 체감 → 가입 전환율 상승

### 2-2. 이메일-only 브리프 구독 퍼널
- `/brief` 페이지 하단 스티키 CTA: "매일 오전 8시, 오늘의 공공조달 공고 요약을 받아보세요"
- 이메일만 입력 → 회원가입 안 해도 구독 가능
- 2주차부터 "회원이면 제안서까지 열람 가능" 후속 메일

### 2-3. 가입 쿠폰 강화
- 현재: WELCOME10K (1만원)
- 제안 A: **"가입 즉시 PDF 1건 무료 다운로드 쿠폰"** (49,000원 상당)
- 제안 B: 추천인 코드로 서로 5천원씩 (바이럴 루프)

---

## 3. 상품 상세 → 구매 전환 후킹

### 3-1. PDF → PPT 업그레이드 유도
- 공식 가격: PPT 99,000원 / PDF 49,000원 / 업그레이드 50,000원
- PDF 상품 상세에 **"지금 5만원만 더하면 PPT까지 받을 수 있어요"** 핑크 박스
- 구매 후 주문 확인 이메일에서도 동일 업그레이드 링크

### 3-2. 번들 할인 (Cross-sell)
- 기술제안서 + 입찰가이드 같이 담으면 15% 할인
- "이 상품을 본 사람이 함께 구매한 상품" 섹션

### 3-3. 후기 신뢰 강화
- 별점 분포 바(5/4/3/2/1 별 비율 시각화)
- 실명 인증 마크 "✓ 구매 확인됨"
- 가장 도움됐다고 투표받은 후기 TOP3 상단 고정

### 3-4. 반환 보증 배지
- "첨부파일 손상 시 100% 환불 · 디지털 상품 특성상 열람 전까지 환불 가능"
- 신뢰 배지: "토스페이먼츠 PG사 정식 계약 · 전자상거래법 준수"

---

## 4. 체크아웃 이탈 방지

### 4-1. 결제 직전 마지막 설득
- 카드 정보 입력 단계에서 "방금 24명이 같은 제안서를 구매했어요" 카운터
- "이 결제를 완료하면 `WELCOME10K` 쿠폰이 자동 적용돼요 (-10,000원)"

### 4-2. 장바구니 이탈 복구 이메일
- 1시간 뒤: "담아두신 제안서, 잊지 않으셨죠?" + 썸네일 + 바로 결제 링크
- 24시간 뒤: 10% 쿠폰 포함 재유도

### 4-3. 게스트 결제 경로 신설
- 로그인 없이 이메일만으로 결제 가능 (이메일로 다운로드 링크 발송)
- 결제 후 "비밀번호만 설정하면 계정이 됩니다" 원터치 가입

## 5. 구매 후 리텐션·LTV 후킹

### 5-1. 주문 완료 페이지에 다음 액션 3가지
1. **다른 공고에도 제안서가 필요하다면?** → 맞춤 공고 알림 설정 CTA
2. **컨설팅 10만원 할인** → 제안서 + 30분 컨설팅 번들
3. **친구 추천** → 양쪽 5천원 쿠폰

### 5-2. 구매 7일 후 리뷰 요청 이메일
- "사용해보시니 어떠셨나요? 리뷰 작성하면 다음 구매 시 20% 쿠폰"
- 리뷰가 다시 소셜 프루프로 순환

### 5-3. 북마크 스냅샷 활용 (2026-04-14 구현됨)
- 마이페이지 북마크에 "관심 공고 마감 3일 전 알림" 토글
- 푸시 알림 또는 이메일로 자동 발송

## 6. SEO·오가닉 유입 후킹

### 6-1. 공고 상세 롱테일 키워드 최적화 (구현됨 2026-04-14)
- Article JSON-LD + canonical + 동적 메타 OK
- 추가 제안: 공고명 + "제안서 작성법" 블로그 자동 생성 (AI로 초안 작성 후 내부 편집)

### 6-2. 일일 브리프를 SEO 자산으로
- `/brief/[slug]` 페이지가 이미 sitemap에 포함됨 ✓
- 추가: 브리프 내 공고명마다 `/announcements/[id]` 링크 → 내부 링크 그래프 강화

### 6-3. 경쟁 키워드 타겟 랜딩 페이지
- "나라장터 기술제안서 샘플"
- "조달청 입찰가이드 다운로드"
- "공공조달 제안서 양식"
- 각각 전용 랜딩에 관련 상품 3~5개 그리드 + CTA

---

## 7. 바이럴·SNS 후킹

### 7-1. 공유 시 쿠폰 지급
- 상품 상세 공유 버튼(카톡/링크 복사) → 공유 완료 시 5천원 쿠폰
- 공유 링크 열람자에게도 동일 쿠폰 자동 부여

### 7-2. 주간 입찰 리포트 카드뉴스
- 매주 월요일 "지난 주 공공조달 하이라이트" 1장짜리 이미지
- 인스타·링크드인 크로스 포스팅

### 7-3. 공공조달 벤더 커뮤니티
- 로그인 사용자 전용 Q&A 게시판
- 답변 채택 시 포인트 → 상품 구매 시 사용

---

## 8. 데이터·개인화 후킹

### 8-1. 업종·지역 기반 맞춤 추천
- 회원가입 시 업종(IT/건설/컨설팅), 주요 지역(서울/부산) 수집
- 홈 진입 시 해당 업종·지역 관련 공고/상품 상단 배치

### 8-2. 열람 이력 기반 리타겟팅
- 페이지 뷰 이력(`page_views` 테이블) 활용
- "최근 본 상품" 사이드 위젯
- 5분 이상 체류했지만 미구매한 상품 → 24시간 후 이메일

### 8-3. LLM 기반 공고 분석 어시스턴트
- 공고 상세에 "이 공고에 참여하려면 어떤 자격이 필요한가요?" AI Q&A
- 응답 하단에 관련 제안서 상품 자동 추천

---

## 9. 신뢰 강화 후킹

### 9-1. 실제 낙찰 사례 갤러리
- "이 제안서로 실제 낙찰된 기업이 30곳" 카운터
- (동의받은) 익명 로고 그리드: "○○테크놀로지가 이 제안서로 12억 규모 사업 수주"

### 9-2. 파트너 기관 로고
- 조달청, 나라장터, KISA, 중소벤처기업부 등 (공식 파트너십 있는 경우만)

### 9-3. 환불 현황 투명 공개
- 푸터 또는 FAQ에 "지난 달 환불 0건" 공개
- 정직함이 강력한 후킹이 됨

## 측정 지표 (전후 비교 필수)

- **회원가입 전환율**: 방문 → 가입 %
- **구매 전환율**: 상품 상세 → 결제 완료 %
- **장바구니 이탈률**: 담기 → 결제 전 이탈 %
- **업그레이드 전환율**: PDF 구매자 중 PPT 업그레이드 %
- **재구매율**: 30일 내 2회 이상 구매 고객 비율
- **이메일 CTR**: 브리프·리마인더 클릭률

각 후킹 배포 전 A/B 테스트 프레임워크(예: GrowthBook 또는 간단한 피쳐 플래그)로 대조군 확보 권장.
