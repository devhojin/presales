# Product Marketing Context

*Last updated: 2026-04-16 (마케팅 전면 개선 세션 반영)*

## Product Overview
**One-liner:** 공공조달 입찰에서 실제 낙찰된 제안서 템플릿을 구매·다운로드할 수 있는 마켓플레이스
**What it does:** 나라장터·조달청 입찰에 참여하는 기업이 실제 낙찰 기록이 있는 제안서(PPT/HWP/PDF)를 즉시 다운로드하여, 자사 내용만 수정해 전문가 수준의 기술제안서를 완성할 수 있게 합니다. 제안서 판매 외에 공고 자동 수집, IT피드, 모닝브리프 이메일, 전문가 컨설팅까지 제공하는 입찰 성공 원스톱 플랫폼입니다.
**Product category:** 공공조달 제안서 마켓플레이스 / 입찰 지원 플랫폼
**Product type:** 디지털 상품 마켓플레이스 + 컨설팅 서비스
**Business model:**
- 제안서 템플릿 개별 판매 (무료 ~ 유료)
- 컨설팅 서비스 3단계: 스팟 상담(15만원/30분), 제안서 리뷰(50만원/건), 프로젝트 컨설팅(300만원~/프로젝트)
- 할인 번들 (교차 상품 할인)

## Target Audience
**Target companies:**
- 공공조달 입찰에 참여하는 중소·중견 IT 서비스 기업
- 정부과제·공공사업에 처음 진출하는 스타트업
- SI/ITS/스마트시티/스마트팩토리/AI 분야 기업
- 사업계획서가 필요한 초기 스타트업

**Decision-makers:**
- 사업개발 팀장/이사
- 중소기업 대표이사
- 경영지원실/관리팀 담당자
- 기술영업(Pre-sales) 담당자

**Primary use case:** 나라장터 입찰 마감 전 짧은 시간 안에 전문가 수준의 기술제안서를 완성하는 것

**Jobs to be done:**
- "처음 쓰는 공공조달 제안서인데, 정답 포맷이 뭔지 모르겠다" → 낙찰된 레퍼런스를 보고 구조를 잡고 싶다
- "입찰 마감 48시간 전인데 제안서를 아직 못 썼다" → 검증된 템플릿에 내용만 바꿔서 빠르게 완성하고 싶다
- "컨설팅 업체에 500만원 맡기는 건 부담이다" → 템플릿으로 직접 해결하고 싶다

**Use cases:**
- ITS(지능형교통시스템) 유지관리 용역 입찰 제안서 작성
- 스마트시티/스마트팩토리 사업 기술제안서 작성
- AI 기반 신사업 정부과제 사업계획서 작성
- 나라장터 평가표 항목별 감점 방지를 위한 구성 참고
- 발표 PT 준비를 위한 발표자료 템플릿 활용

## Problems & Pain Points
**Core problem:** 좋은 기술력을 가진 기업이 제안서 작성 능력 부족으로 공공조달 입찰에서 탈락한다

**Why alternatives fall short:**
- 컨설팅 업체: 건당 500만원+ 고비용, 일정 조율 필요, 자사 인력 역량 축적 안 됨
- 인터넷 검색: 실제 낙찰된 제안서는 공개되지 않음, 나온 것들은 일반적인 양식 수준
- 자체 작성: 공공조달 특유의 암묵적 "정답 포맷"을 모르면 내용이 좋아도 읽히지 않음
- 동종업계 지인: 경쟁 관계라 실제 낙찰 제안서를 공유하지 않음

**What it costs them:**
- 기술점수 70점 미만 → 가격이 아무리 낮아도 자동 탈락
- 한 번 탈락하면 다음 입찰까지 수개월 대기
- 입찰 준비에 2~4주 소요 (핵심 인력의 기회비용)
- 컨설팅 외주 시 건당 500만원 이상

**Emotional tension:**
- "마감까지 48시간. 다 쓴 줄 알았던 제안서를 처음부터 다시 엽니다."
- "이번에도 떨어지면 어떡하지—"
- 기술력은 자신 있는데 '문서 때문에' 진다는 좌절감

## Competitive Landscape
**Direct:** 국내에 동일한 "낙찰 제안서 마켓플레이스"는 현재 없음 — 블루오션
**Secondary:**
- 제안서 컨설팅 업체 (건당 500만원+) — 고비용, 스케일 불가
- 나라장터/조달청 공식 자료 — 양식만 제공, 실전 제안서 없음
- 문서 템플릿 마켓 (미리캔버스, 크몽 등) — 공공조달 전문성 없음
**Indirect:**
- 사내 제안서 팀 보유 대기업 — 중소기업에게는 현실적으로 불가능
- 프리랜서 제안서 작성 대행 — 품질 편차 크고, 노하우 축적 안 됨

**How each falls short:** 모두 "실제 낙찰된 제안서 원본"에 접근할 수 없다는 공통 한계

## Differentiation
**Key differentiators:**
- 실제 나라장터에서 낙찰된 제안서 원본 (감이 아닌 실적 검증)
- PPT/HWP/PDF 원본 파일 즉시 다운로드 (편집 가능)
- 6개 전문 분야별 맞춤 구성 (기술제안, 입찰가이드, 발표자료, 가격제안, 풀패키지, 사업계획서)
- 부가 생태계: 공고 자동 수집 + IT피드 + 모닝브리프 + 전문가 컨설팅
- 결제 후 5초 안에 다운로드 — 급한 입찰에 즉시 대응

**How we do it differently:** 공공조달 전문가 팀(ITS/스마트시티/AI/금융SI 실적 보유)이 직접 검증한 낙찰 제안서를 디지털 상품으로 패키징
**Why that's better:** 컨설팅 1/10 비용으로 동일한 품질의 출발점을 확보, 자사 인력이 직접 학습·역량 축적 가능
**Why customers choose us:** "기출문제"에 해당하는 실제 낙찰 제안서를 볼 수 있는 유일한 곳

**Anti-persona:**
- 이미 전담 제안서 팀(10인 이상)을 보유한 대기업
- 공공조달이 아닌 민간 B2B 입찰만 하는 기업
- 제안서 없이 수의계약으로만 수주하는 기업

## Switching Dynamics
**Push:** 입찰 마감 임박한데 제안서 완성도가 낮음 / 이전 입찰 탈락 경험 / 컨설팅 비용 부담
**Pull:** 실제 낙찰 기록이 있는 제안서를 바로 볼 수 있음 / 5초 만에 다운로드 / 무료 샘플로 위험 없이 시작
**Habit:** "우리 팀이 늘 쓰던 양식이 있어서" / "지인에게 물어보는 게 익숙해서"
**Anxiety:** "템플릿을 써도 결국 탈락하면?" / "경쟁사도 같은 템플릿을 쓰면?"
