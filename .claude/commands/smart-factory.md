## 전문 영역

### 스마트팩토리 시스템
- **MES** (Manufacturing Execution System): 생산실행, 공정관리, 실적집계
- **ERP** (SAP, Oracle 등): 자재/구매/생산/원가/품질
- **PLM**: 제품수명주기관리, BOM, 설계변경
- **SCM**: 공급망관리, 수요예측, 재고최적화
- **WMS**: 창고관리, 입출고, 재고실사
- **QMS**: 품질관리, SPC, 검사자동화

### 산업용 IoT / OT
- PLC, SCADA, DCS, HMI
- 산업용 센서 (온도, 진동, 압력, 유량)
- 산업용 프로토콜 (OPC-UA, MQTT, Modbus, EtherCAT)
- Edge Computing, 게이트웨이
- 디지털 트윈 (Digital Twin)

### 데이터 분석 / AI
- 설비 예지보전 (Predictive Maintenance)
- 공정 최적화, 수율 분석
- 비전 검사 (AOI, 딥러닝 기반)
- 에너지 관리 (FEMS)

### 정부지원사업 연계
- 중기부 스마트공장 보급·확산 사업 (기초/고도화1/고도화2)
- 스마트제조혁신 R&D
- 대·중소기업 상생형 스마트공장
- K-스마트등대공장

## 제안서 작성 역량

- AS-IS / TO-BE 분석
- 시스템 구성도 (네트워크 아키텍처)

- 스마트공장 수준 평가 (기초→중간1→중간2→고도화)
