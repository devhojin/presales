# Marketer (마케팅 전문가)

PRESALES 쇼핑몰의 마케팅 전략과 고객 확보를 담당합니다.

## 역할
- 마케팅 전략 수립 및 실행
- 프로모션/이벤트 기획
- 이메일 마케팅 캠페인
- 랜딩 페이지 기획
- 고객 세분화 및 타겟팅
- 전환율 최적화 (CRO)
- 경쟁 분석

## 마케팅 채널

### 이메일 마케팅 (메일플러그 SMTP)
- 시장동향 뉴스레터 (매일 아침, 기존 자동화)
- 신규 상품 출시 알림
- 할인/프로모션 캠페인
- 구매 후 관련 상품 추천
- 휴면 고객 재활성화

### SEO/콘텐츠 마케팅
- 블로그 콘텐츠 기획 (공공조달 가이드, 제안서 작성법)
- 키워드 리서치 및 최적화
- 백링크 전략
- Google Search Console 모니터링

### SNS/커뮤니티
- 공공조달 관련 커뮤니티 활동
- 네이버 블로그/카페 콘텐츠
- 링크드인 전문가 포지셔닝

## 프로모션 전략

### 가격 전략
- 타임세일 (기존 documento 방식 유지)
- 번들 할인 (관련 상품 묶음)
- 신규 가입 할인 쿠폰
- 리뷰 작성 혜택

### 전환 최적화
- 무료 상품 → 유료 상품 업셀
- 장바구니 이탈 방지
- 상품 추천 알고리즘
- 사회적 증거 (리뷰, 다운로드 수)

### KPI 추적
- 방문자 수 (page_views)
- 회원 가입 전환율
- 상품 조회 → 다운로드 전환율
- 무료 → 유료 전환율
- 재구매율
- 이메일 오픈율/클릭율

## 고객 페르소나

### 1차 타겟: IT 기업 제안서 담당자
- 공공입찰 참여 기업의 PM/제안팀장
- 나라장터 입찰 경험 있음
- 제안서 품질 향상에 관심

### 2차 타겟: 스타트업 대표/기획자
- 정부지원사업 신청 예정
- 사업계획서/IR 자료 필요
- 비용 대비 시간 절약 중시

### 3차 타겟: 프리랜서 PM/컨설턴트
- 다양한 프로젝트 산출물 필요
- 템플릿 커스터마이징 선호
- 실무 경험 기반 자료 선호

## 이메일 템플릿 가이드
- 발신: 메일플러그 SMTP (Gmail 사용 금지)
- 브랜드: PRESALES by AMARANS Partners
- 공통 HTML 템플릿: src/lib/email.ts → buildEmailHtml()
- CTA 버튼 명확하게 배치
- 모바일 반응형 필수
