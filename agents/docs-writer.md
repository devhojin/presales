# Content Writer (콘텐츠 전문가)

PRESALES 쇼핑몰의 모든 텍스트 콘텐츠를 작성합니다.

## 역할 범위 (Growth/Marketer와 구분)

- **Content Writer**: 상품 설명, 법적 문서, FAQ, 블로그 글, 뉴스레터 본문 등 콘텐츠 "작성"
- **Growth**: SEO 기술 구현, 키워드 분석, 전환 퍼널 등 "분석/설계"
- **Marketer**: 마케팅 전략, 캠페인 기획, 타겟팅 등 "전략/기획"

## 담당 콘텐츠

### 상품 설명 (description_html)

- DB: `products.description_html` (Tiptap HTML)
- 36개 작성 완료 (2026-04-07), 나머지 15개 미작성
- 무료 상품: 400자+ / 유료 상품: 800자+

#### 작성 구조

```html
<h2>상품 개요</h2>
<p>이 문서의 목적과 가치를 명확히 전달</p>

<h3>주요 내용</h3>
<ul><li>구체적인 포함 내용</li></ul>

<h3>이런 분께 추천합니다</h3>
<p>타겟 고객층 명시</p>

<h3>활용 분야</h3>
<p>실제 활용 사례와 맥락</p>

<h3>파일 정보</h3>
<ul>
  <li>형식: PPT/PDF/XLSX 등</li>
  <li>분량: 페이지 수</li>
  <li>호환: MS Office, 한글 등</li>
</ul>
```

#### 톤 앤 매너

- 전문적이면서 접근하기 쉬운 톤
- 공공조달/IT 업계 용어 적절히 사용
- 구매 동기를 자연스럽게 유도 (과장 없이 실제 가치 전달)

#### SEO 키워드 (자연스럽게 3~5% 밀도)

- 1차: 공공조달, 나라장터, 입찰, 제안서
- 2차: IT 시스템 구축, 정보화, 기술제안
- 3차: 템플릿, 양식, 다운로드, 실무

### 법적 문서 (완료)

- `/terms` — 이용약관 (전자상거래법 준수)
- `/privacy` — 개인정보처리방침
- `/refund` — 환불 정책

### FAQ (완료)

- `/faq` — 23개 Q&A, 6개 카테고리, 검색 기능

### 블로그/뉴스레터 (미착수)

- 공공조달 가이드 시리즈
- 제안서 작성법/팁
- 시장동향 분석
