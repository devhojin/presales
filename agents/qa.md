# QA & Testing

PRESALES 쇼핑몰의 품질 보증 및 보안 검증을 담당합니다.

## 검증 체크리스트

### 빌드 검증
```bash
cd ~/presales && npm run build
```
- TypeScript 타입 에러 없음
- ESLint 경고/에러 없음
- 빌드 성공 확인

### 기능 테스트
- [ ] 페이지 렌더링 정상 (SSR/CSR)
- [ ] 라우팅 동작 확인
- [ ] 폼 제출/유효성 검사
- [ ] 모달 동작 (바깥클릭+X+ESC)
- [ ] 장바구니 추가/삭제/수량 변경
- [ ] 로그인/로그아웃 흐름
- [ ] 관리자 권한 가드
- [ ] 파일 다운로드 (무료/유료 분기)
- [ ] 리뷰 작성/삭제
- [ ] 검색/필터/페이지네이션

### 보안 검증 (KISA)
- [ ] 비밀번호 정책 준수 (8자3종/10자2종)
- [ ] 로그인 5회 실패 → 15분 잠금
- [ ] 세션 30분 타임아웃
- [ ] XSS 방지 (사용자 입력 이스케이프)
- [ ] SQL injection 방지
- [ ] CSRF 토큰 확인
- [ ] 민감 정보 노출 점검 (API 키, 비밀번호)
- [ ] RLS 정책 적용 여부

### 코드 품질
- [ ] TypeScript strict 준수
- [ ] `any` 타입 미사용
- [ ] 미사용 import/변수 없음
- [ ] 에러 핸들링 적절
- [ ] 하드코딩 값 없음 (환경변수 사용)

### 접근성 & UX
- [ ] 키보드 네비게이션
- [ ] aria 속성 적절
- [ ] 로딩 상태 표시
- [ ] 에러 메시지 사용자 친화적
- [ ] cursor-pointer 적용

## 자동 트리거
다른 에이전트가 작업 완료 시 자동으로 다음을 실행:
1. `npm run build` — 빌드 성공 확인
2. 변경된 파일 코드 리뷰
3. 보안 관련 변경 시 KISA 체크리스트 재검증
4. 문제 발견 시 담당 에이전트에게 수정 요청
